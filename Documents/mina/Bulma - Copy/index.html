Mis asi on Bulma?

"Modernne, Javascripti-vaba CSS raamistik, mis põhineb Flexboxil."

Flexbox

Moodne paigutus ilma floatideta.

No JS

Ainult stiilid, loogika teed ise.

Mobile

Töötab telefonis ideaalselt.

02 / 09

Ajalugu

  • 2016
    Loodud Jeremy Thomase poolt.
  • Open Source
    Üks populaarsemaid CSS raamistikke GitHubis.

Filosoofia

✓ Lihtsus: Ingliskeelsed klassinimed.

✓ Puhtus: Ei mingeid ülearuseid stiile.

✓ Modulaarsus: Kasuta ainult seda, mida vajad.

03 / 09

Kuidas see töötab?

SASS Muutujad

Muuda värve ja fonte ühe reaga:

$primary: #8c67ef;
$family-sans-serif: "Nunito", sans-serif;
@import "bulma";

Loetavad klassid

Klassinimed on loogilised:

is-primary is-light is-rounded is-loading
