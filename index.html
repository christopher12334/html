01 / 09

Mis asi on Bulma?

"Modernne, Javascripti-vaba CSS raamistik, mis põhineb Flexboxil."

Flexbox

Moodne paigutus, mis kohandub ekraaniga.

No JS

Ainult stiilid. Ei vaja töötamiseks skripte.

Mobile-First

100% responsiivne igas seadmes.

02 / 09

Ajalugu

  • 2016
    Loodud Jeremy Thomase poolt alternatiivina Bootstrapile.
  • Open Source
    Üle 45 000 tärni GitHubis ja suur kogukond.

Disainifilosoofia

✓ Lihtsus: Ingliskeelsed ja loetavad klassinimed.

✓ Puhtus: Minimalistlik ja moodne välimus.

✓ Modulaarsus: Impordi ainult see, mida vajad.

03 / 09

Kuidas see töötab?

SASS Muutujad

Kogu disaini saab muuta paari reaga:

$primary: #8c67ef;
$family-sans-serif: "Nunito", sans-serif;
@import "bulma";

Loetavad klassid

Klassinimed kirjeldavad täpselt sisu:

is-primary is-light is-rounded is-loading
